# Changelog

All notable changes to this project are documented in this file.

The format is based on Keep a Changelog.

## [0.3.0] - 2026-05-01

### Added
- Replay CLI with tenant/date-range filtering and controlled replay rate.
- Chaos suite scripts for dependency outages and operational stress scenarios.
- Benchmark suite with regression comparison tooling.

## [0.2.0] - 2026-05-01

### Added
- Ingest API with schema validation, archive-first flow, and middleware instrumentation.
- Worker service with idempotency handling and outbox publisher.
- Helm and raw Kubernetes deployment manifests.

## [0.1.0] - 2026-05-01

### Added
- Project bootstrap, config loading, core event model, and partitioned database schema.
- Initial CI workflows, GitHub templates, and Grafana dashboard provisioning.
